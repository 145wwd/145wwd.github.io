🎯 十神信息之象很重要，要到达命理高层次必须深通十神信息之象，通过十神之象来推断一个人富贵贫贱，才能更精确地推断出一个人真实运行轨迹。

心性分析 · 信息之象 · 性格推断 · 人生轨迹

深通十神心性，精准推断命运

一、十神信息之象的意义

1. 从心性推断人生

通过十神反应的信息之象，可以初步判断一个人的一生。

例一：一个人十神反应的信息之象是：为人循规蹈矩、胆小怕事，从不愿出头露面。

推断：

  • 为人循规蹈矩、胆小怕事
  • 自然不敢做违法之事
  • 也不适宜做生意（较胆小怕赔）
  • 做事不愿出头露面
  • 到哪也显示不出才能

结论：只适合给人打工、挣工资，富贵自然难谈。魄力没有，自然也难为官，只是一个平平常常的靠打工、挣工资维生的人。

2. 从组合推断吉凶

例二：一个人身旺，透劫财与伤官。

信息之象：

  • 好侵犯人
  • 好惹事生非
  • 胆大、目无法
  • 好吃喝、嫖赌

推断：

  • 不是社会上的赌徒、流氓、违法乱伦分子
  • 遭恶运定会做出违法犯罪之事
  • 必然是狱中常客
  • 一生多官非口舌

二、正官心性信息之象

1. 正官的基本心性

心性 说明
有责任感 循规蹈矩
正官太旺、身太弱 胆小怕事
正官太弱、身太强 放荡不羁、难以领导

2. 正官为喜用

心性 说明
有上进心 对工作责任心强
工作有条有理 按部就班

3. 正官在四柱

位置 信息之象
年柱为喜用 在兄弟姐妹中多受父母爱护
年干支皆正官，身弱 父母管得严，压力大，自立更生
月柱为正官喜用 受父母疼爱，一生少辛苦
日支正官 应变能力强，逢财运可发大财

4. 特殊情况

情况 信息之象
官杀同透（女命） 婚姻多不顺，多夫之象
身弱女命日坐正印 依靠性强，做事有些慢
正官坐羊刃 一生多辛苦

三、七杀心性信息之象

1. 七杀的基本心性

心性 说明
七杀有制 可转为正官心性
正官多现 又转为七杀的心性

2. 正官与七杀的区别

十神 特点
正官 有约身引善之功能
七杀 主霸道不讲理

3. 七杀旺的心性

心性 说明
七杀旺身旺 说话压碴，做事有魄力
有感召力 有管理才能
脾气不好 多蛮横
走正路 能成就一番事业
走黑道 成为黑社会之头目

4. 七杀的其他心性

心性 说明
好猜疑 压力大，晚上好做噩梦
主聪明伶俐 智商高

5. 女命七杀

情况 信息之象
七杀重身旺 和丈夫不和，对外蛮横不讲理
有时 做事干脆，果断，有条理能干

6. 七杀在四柱

情况 信息之象
身弱杀旺 经常遇到小人的捉弄
身强杀无制 常成为别人的小人，好捉弄人
七杀坐日支 配偶性格暴躁，倔强
七杀在时柱 子女多不孝

四、正财心性信息之象

1. 正财的基本心性

心性 说明
精神含义 小态，重视感观的暗示和欲望的反映
注重 具体和供职的物象和事物
占有欲 有追求功名思想
财重 对神秘文化宗教信仰都差

2. 正财为喜用

心性 说明
财弱财少 比较节俭，或说比较抠门

3. 正财与婚姻

情况 信息之象
日支财逢空亡 男的易晚婚
日支合正财 夫妻恩爱（男命）
日支合非正财 妻子有外遇
正财坐羊刃 有破财之象或妻子不贤

4. 正财在四柱

位置 信息之象
月柱 勤俭节约
时柱 子女富有，可从商
坐死墓绝地 妻子有大病，身体不好

5. 特殊情况

情况 信息之象
正财劫财同现 一生易遭小人破财
正财多不论喜忌 好逸恶劳，贪婪
女命身弱财多财旺 主风流

五、偏财心性信息之象

1. 偏财的基本含义

含义 说明
外道来的财 受贿、中彩票、赌博、送礼之类不劳而获之财
偏财藏地支多 暗生官，受贿之象

2. 偏财的心性

心性 说明
重视现实 从现实出发，没太深的偏见
不太亲密的妻子 不太重视日常家务用具
对钱物 抱着可有可无的态度
比较不专一 与正财比较谨慎专一相对

3. 偏财为用

心性 说明
为人慷慨 重义轻财，人缘极佳
为忌 荣华虚浮，贪图享受很懒

4. 偏财在四柱

情况 信息之象
天干太旺 有好酒色之象
透两个偏财 婚姻不顺，有正妻还应有个情人
坐死墓绝 有父或妾哀困之感
年干偏财年支比劫 不利于在他乡求财
年柱干支皆偏财 幼年为养子之象
日支偏财 男命易有风流之事
时柱无刑冲比劫 晚年好一点

5. 女命偏财

情况 信息之象
偏财多为忌神 受父亲拖累

六、伤官心性信息之象

1. 伤官的基本心性

心性 说明
愿出风头 喜欢出名
坚持自己的言论 好与官做对
靠口输出 重视别人对自己的掌声
适宜从事 文艺歌星
重视回报 希望得到感激

2. 伤官旺的心性

心性 说明
不怕官 胆大妄为
注重名声 注重别人怎么评价自己
好大喜功 喜欢吹牛，好夸张
个性太放荡 目无尊长，没有礼貌
女的多半 作风不正

3. 伤官的其他心性

心性 说明
好奇心强 有夸张力，幻想
非常罗曼蒂克 富于想象力
反叛心理 好胜心强好逞强

4. 女命伤官

情况 信息之象
日坐伤官 婚姻不顺，看谁都不如自己
心高气傲 自负清高

5. 伤官在四柱

位置 信息之象
年柱 祖业漂流
年柱逢岁运加力 容易有灾
月柱 有伤手足或手足失和之象
日支 男命克妻，女命克夫
时柱 男命子女缘薄

七、食神心性信息之象

1. 食神的基本心性

心性 说明
不喜欢独裁 愿意民主
行事做决定 不喜欢强加于人
有意识流动聪明 与伤官类似

2. 食神的代表

代表 说明
说话 动身、旅游、跳舞、运动
趋于平淡 与伤官好出风头相对

3. 食神透出

心性 说明
口福好 愿意吃红烧肉
消化器官好 主长寿

4. 食神在四柱

情况 信息之象
食伤太多 好逸恶劳，不愿干活
食神逢空或被枭夺 小时候缺奶吃
食神与七杀同透 有掌权之象
月支食神 为人和气
月支食神月支正官 有官运，适宜挣工资

5. 女命食神

情况 信息之象
食神多 婚姻不顺
食神与偏印同柱 易有流产之事

八、正印心性信息之象

1. 正印的基本心性

心性 说明
爱情奉献 有爱心
忍辱负重 勤劳勤恳耐劳

2. 正印的负面

心性 说明
随大流 没有主见
印枭多 做事虎头蛇尾，主意好变
没有压力感 平时懒惰
容易知足 什么都不是搞得太复杂

3. 正印的代表

代表 说明
权利 地位、事业、靠山
主文 名誉

4. 正印与其他

情况 信息之象
官无印不显 身旺胜官，无印掌权不实
财旺行印运 一般是身弱，名声不好
印旺之人 依靠性大
正印坐七杀 杀印相生，诚实守信用
正印坐枭 缺乏果断力

5. 女命正印

情况 信息之象
多正印 克夫之象

九、偏印心性信息之象

1. 偏印的基本心性

心性 说明
比较呆板 呆滞，不愿想办法
不愿侵犯别人 也不愿关心别人
有时有修养 但有时不通人情

2. 偏印与正印的区别

十神 心性
正印 愿意付出
偏印 不情愿付出

3. 偏印透出

心性 说明
心眼多 有时怪兮兮的

4. 偏印的代表

代表 说明
非正流行业上的权力 艺术、医生、律师、宗教、自由业、服务业

5. 偏印为用

心性 说明
精明能干 鬼点子多

6. 偏印在四柱

情况 信息之象
偏印旺 不利子女
偏印+官杀混杂 多成多败之命
偏印坐比肩 一生多劳苦
干支皆偏印或透两个偏印且旺身又旺 家中常丢东西
年柱见偏印为忌 被祖业，损家名
日支偏印为忌 男不得良妻，女不得良夫

7. 女命偏印

情况 信息之象
时柱偏印 不利子女，多有流产、病伤亡现象
偏印多 易有流产之事

十、比肩劫财心性信息之象

1. 比肩的基本心性

心性 说明
不愿主动侵犯别人 不愿惹祸生非
自尊心强 要面子
为人固执 不易变通
不愿让别人管 喜欢自由

2. 劫财的基本心性

心性 说明
身旺劫财透 好侵犯别人，好惹灾祸，惹是非

3. 比劫在四柱

情况 信息之象
比劫多 不利于合伙生意，财易被夺
四柱多比肩无制 兄弟不和，太刚
比肩在年柱 有独立分家之象
日支比肩 婚姻不顺，不论男女
比肩坐死墓绝地 兄弟姐妹一般不在一起
无比劫之人身又旺 性格古怪，不合群
比劫多 易有破财克妻之事，有钱也难存住

十神心性运用的核心：通过十神的信息之象可以推断一个人的性格、行为、命运。十神心性是断命的重要依据。
