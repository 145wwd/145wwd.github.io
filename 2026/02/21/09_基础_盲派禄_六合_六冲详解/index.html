盲派禄、六合、六冲详解：做功的重要方式

玄机阁主2026-02-212026-02-21

⚡ 盲派禄、六合、六冲详解

🎯 禄、六合、六冲是盲派命理做功的重要方式，掌握这些内容是断命的关键。

一、禄的详解

1. 什么是禄？

禄是八字中很重要的东西，代表日主的身体、俸禄、财富。

2. 禄的对应关系

盲师一派讲干禄：

天干 禄 说明
甲 寅 甲禄在寅
乙 卯 乙禄在卯
丙 巳 丙禄在巳
丁 午 丁禄在午
戊 巳 戊禄在巳
己 午 己禄在午
庚 申 庚禄在申
辛 酉 辛禄在酉
壬 亥 壬禄在亥
癸 子 癸禄在子

3. 特殊禄关系

关系 说明
丙与戊 同禄在巳
丁与己 同禄在午
丁与未 半个禄
己见丑 半个禄
癸与丑 半个禄

4. 禄的作用

作用 说明
代表身体 禄是日主的化身
代表财富 八字不见财星以禄当财
代表俸禄 官禄、财禄

5. 禄的案例

案例一：蒋介石命造

分析：

  • 年柱丁亥，合局中财官
  • 丁亥自合，是去而再合
  • 丁是己的禄，是自己

结论：妙在丁亥自合，为我所用，所以能当总统。

案例二：以禄当财

分析：

  • 局中不见财星
  • 以年支巳禄当财
  • 年上为国外

结论：此妇庚辰年出国，在国外财运很好。

案例三：禄与婚姻

分析：

  • 己见丁是禄
  • 丁丑年，丁见丑是财库
  • 巳丑合妻宫

结论：丁亥大运丁丑流年财运大好，同年结婚。戊寅年离婚，是局中寅巳空的应期。

二、六合的详解

1. 六合的基本概念

六合：子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合。

2. 六合的五种用法

（一）合去

含义：某一字在八字中太弱，合即被去，天干地支同论。

案例：

分析：

  • 乙木印星在局中太弱
  • 乙庚合则去，克母之象
  • 庚戌年是庚到的应期

结论：丙戌大运庚戌流年死母亲。

（二）合留

含义：流年与八字或大运中某字合为合留，留为得到的意思，婚姻应居多。

案例：

分析：

  • 庚辰年结婚
  • 辰与酉合妻宫

结论：此应辰与酉合妻宫，合留而得妻。

（三）合动

含义：八字大运中某一天干或地支临旺地，原局中静而不合，逢太岁合时为合动，主被合动的一方在此年应事。

案例一：

分析：

  • 庚辰年提镇长
  • 乙庚合动官星

结论：乙庚合动官星之故。

案例二：

分析：

  • 此女庚辰去美国
  • 乙庚合动门户

结论：乙庚合动门户之故。

（四）合绊

含义：合绊双方失去作用。

案例：

分析：

  • 壬水伤官被丁火合绊
  • 伤官代表子女

结论：此女结婚十几年不生孩子。

（五）合闭

含义：合闭库门。

案例一：

分析：

  • 走丁巳大运财官两旺
  • 乙卯运借债度日
  • 卯戌合闭财门

结论：因卯戌合闭财门之故。

案例二：刘晓庆命造：

分析：

  • 辛卯大运未年破财官司
  • 卯戌合闭财门
  • 甲申年挣钱，申把卯合住

结论：申把卯合住，不去合戌了，所以挣钱。

3. 六合用法总结

用法 含义 应事
合去 弱者被合走 去掉、失去
合留 合住而得到 婚姻、得到
合动 静者被合动 应事、变动
合绊 双方失去作用 受阻、停滞
合闭 合闭库门 财门关闭

三、六冲的详解

1. 六冲的基本概念

六冲：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲。

2. 六冲的应事原理

情况 应事
八字原局有冲 原局自带冲象
大运与八字冲 运程引发冲象
流年与大运冲 流年引发冲象

六冲多应冲动之意，弱则冲去，旺则冲动。

3. 合中逢冲是应期

八字原局中六合或地支暗合，逢冲流年应事，合中逢冲是应期。

4. 六冲案例

案例一：冲则动

分析：

  • 癸未年去远方上学
  • 从济南去青岛
  • 丑未冲之应期

结论：冲则动。

案例二：冲解合

分析：

  • 局中午亥暗合
  • 己巳年冲亥解了合

结论：己巳年结婚，因冲解了合。

案例三：冲升官

分析：

  • 乙酉年升为副市长
  • 酉冲卯

结论：酉冲卯之故，冲去官星之忌。

案例四：冲私奔

分析：

  • 此女谈恋爱，父母不同意
  • 甲申年合男朋友私奔
  • 局中寅丑暗合
  • 月上申冲寅，是父母不同意的原因

结论：申年冲寅的应期，月上为父母。

案例五：冲羊刃

分析：

  • 壬午年腿伤
  • 局中子丑合
  • 午年冲羊刃

结论：冲羊刃主有伤。

5. 六冲用法总结

用法 含义 应事
弱则冲去 弱者被冲掉 去掉、失去
旺则冲动 旺者被冲动 变动、走动
冲解合 冲开合局 解脱、应期
冲羊刃 冲克羊刃 伤灾、手术

四、禄、六合、六冲的关系

1. 做功方式

方式 特点
禄 代表自己，可当财看
六合 合则能去，做功方式
六冲 冲则动，应期方式

2. 相互配合

五、实战要点

1. 禄的要点

要点 内容
禄是身体 代表日主本人
禄可当财 不见财星以禄当财
禄怕冲破 禄被冲破主身体有灾

2. 六合的要点

要点 内容
分清用法 合去、合留、合动、合绊、合闭
看力量 弱者合去，旺者合动
定应期 合留为得到，合闭为关闭

3. 六冲的要点

要点 内容
冲则动 主变动、走动
弱则去 弱者被冲掉
冲解合 冲开合局为应期

总结

核心要点

内容 要点
禄 代表身体、财富
六合 合去、合留、合动、合绊、合闭
六冲 冲动、冲去、冲解合
