盲派命理技法简介：历代相传的断命绝活

🎯 技法是盲派命理最难的部分，主要包括盲人历代相传的一些口诀，是盲派命理的绝活。

一、什么是技法？

技法的定义

技法是盲人命师在多年的断命生涯中形成的经验总结，以口诀的形式代代相传。

技法的特点

特点 说明
口诀形式 简洁易记
经验总结 来自实战
准确度高 运用精准
传承性强 代代相传

技法与理法、象法的关系

法则 层次 作用
理法 基础 判断吉凶
象法 进阶 断具体事项
技法 高级 提高精准度

⚠️ 有时候并说不出它的具体道理来，但运用起来却十分准确。

二、常见技法口诀

1. 财官相关口诀

口诀 含义
制去财与财的原神是发大财的人 财被制住，能发大财
制去官与官的原神是当官的人 官被制住，能当官
制去杀与杀的原神是有权的人 杀被制住，有权势
坐下财库喜刑冲 日支坐财库，喜刑冲打开

详解：

制去财与财的原神：

  • 财星被制住 → 能得到财
  • 财的原神被制 → 财源被控制
  • 两者都被制 → 发大财

坐下财库喜刑冲：

  • 日支为财库（辰戌丑未）
  • 需要刑冲才能打开
  • 打开后能得财

2. 月令相关口诀

口诀 含义
甲生酉月喜水润 甲木生于酉月，喜水来滋润
乙生酉月宜火攻 乙木生于酉月，宜火来克制

详解：

甲生酉月喜水润：

  • 甲木为阳木，生于酉月（秋季）
  • 金旺木死，需要水来通关
  • 水能泄金生木，为用神

乙生酉月宜火攻：

  • 乙木为阴木，生于酉月
  • 七杀当令，需要制杀
  • 火能制金，为用神

3. 职业相关口诀

口诀 含义
外食内杀是当教师的人 食神在外，七杀在内，适合当教师
官印相生是当官的人 官生印，印生身，适合当官
食神制杀是有权的人 食神制七杀，有权势

详解：

外食内杀是当教师的人：

  • 食神在天干（外）：表达、传授
  • 七杀在地支（内）：管束、威严
  • 组合起来：有威严的传授者 → 教师

4. 六亲相关口诀

口诀 含义
年柱逢冲，父母有损 年柱被冲，父母有灾
月柱逢冲，兄弟有变 月柱被冲，兄弟有变故
日柱逢冲，婚姻不顺 日柱被冲，婚姻有问题
时柱逢冲，子女有损 时柱被冲，子女有灾

三、技法的运用领域

1. 判断六亲

要看一个人的父母有没有克，或是否健在：

步骤 方法
第一步 用理法判断吉凶
第二步 用象法看具体象
第三步 用技法定夺

2. 判断婚姻

分析一个八字的婚姻情况：

问题 技法运用
婚姻质量不高 看日支的刑冲合害
会有几次婚姻 看官杀/财星的个数和位置
终生没有婚姻 看配偶宫和配偶星的状态

3. 判断子女

问题 技法运用
有无子女 看时柱和食伤
子女多少 看食伤的个数
子女优劣 看时柱的吉凶

4. 判断事业

问题 技法运用
适合什么职业 看十神的组合
能否当官 看官杀的状态
能否发财 看财星的状态

四、盲派六大要诀

盲派命理有六大要诀，是断命的核心方法：

1. 干支配置分析命局

内容 说明
天干 外在、表现、动
地支 内在、根基、静
配置 干支的组合关系

运用：通过干支配置来分析命局的结构和特点。

2. 宾主理论（宫位从宾主入手）

内容 说明
主 日主、日支
宾 日主以外的干支

运用：区分什么是自己的，什么是别人的。

3. 十神活论（十神意向包括含义的理论）

内容 说明
十神含义 每个十神的象义
十神意向 十神想要做什么

运用：通过十神的含义和意向来断事。

4. 虚实喜忌（天干为虚，地支为实）

内容 说明
虚 天干无根，力量弱
实 天干有根，力量强

运用：判断天干的力量强弱。

5. 应期理论（大运流年）

内容 说明
大运 十年一步运
流年 每年一运

运用：判断吉凶发生的具体时间。

6. 看八字之象

内容 说明
干支象 天干地支的形象
宫位象 四柱位置的含义
十神象 十神的象义

运用：通过看象来断具体事项。

五、六大要诀的关系

综合运用

六大要诀不是孤立的，需要综合运用：

要诀 作用
干支配置 基础框架
宾主理论 定位分析
十神活论 内容解读
虚实喜忌 力量判断
应期理论 时间预测
看八字之象 具体断事

六、技法学习要点

1. 熟记口诀

首先要熟记各种口诀：

  • 财官相关口诀
  • 月令相关口诀
  • 职业相关口诀
  • 六亲相关口诀

2. 理解原理

虽然口诀有时候说不出具体道理，但还是要尽量理解：

  • 为什么这个口诀有效？
  • 口诀背后的原理是什么？
  • 在什么情况下适用？

3. 多练多验证

通过大量案例来验证口诀：

  • 看口诀在实际中是否准确
  • 总结口诀的适用条件
  • 积累实战经验

4. 灵活运用

口诀不能死板套用：

  • 要结合具体八字
  • 要结合理法和象法
  • 要灵活变通

七、技法实例

案例一：断财运

技法运用：

  • 财星壬水三透
  • 财多身弱
  • 无比劫帮身

口诀：财多身弱，富屋贫人

结论：看得到财，得不到财，一生劳碌。

案例二：断官运

技法运用：

  • 七杀甲木透干
  • 食神丙火制杀

口诀：食神制杀，有权有势

结论：有官职，有权势。

总结

技法内容一览

内容 要点
口诀 历代相传的经验总结
六大要诀 干支配置、宾主、十神、虚实、应期、象
运用领域 六亲、婚姻、子女、事业

技法的核心

技法的核心：通过口诀和经验，提高断命的精准度。

技法是盲派命理最难的部分，也是最精华的部分，需要在实践中不断积累和验证。
