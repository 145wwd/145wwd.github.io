正偏财类象详解：财富与婚姻

玄机阁主2026-02-212026-02-21

💰 正偏财类象详解

🎯 看一个人的贫富，要围绕财星来看，看财星为喜、为忌的情况。

一、正偏财的基本含义

1. 正财与偏财

类型 含义
正财 工资、存款、妻子（男命）
偏财 投资、意外之财、父亲、情人

2. 正偏财的共同特点

特点 说明
我克之神 代表占有、控制
代表财富 钱财、物质
男命妻子 配偶信息

二、财富类象

1. 财富判断原则

情况 财富状况
用财见财 富
用财见财受制 有起伏
用财见财逢生 大富
忌财无财 富
忌财见财受制 可富，但有起伏

2. 贫穷判断

情况 说明
忌财见财无制 贫，逢生更贫
用印财破印 懒而贫
忌印财不制印 懒而贫
用官财不生官 懒而贫
忌官财去生官 贫而灾
用食伤财泄食伤 愚而贫
身旺比劫夺财无食伤通关 贫克父

3. 格局与财富

格局 财富状况
从财格+财得食伤生 大富
从财格+无食伤生 小富
从强无财 富
从强见财 贫
从官见财 富
从官无财 不大贵亦不大富
从儿格无财 富
从儿格见财 一般
从弱见财 富

三、婚姻类象

1. 婚姻判断原则

男看财星，女看官星，结合宫位，参看食伤。

2. 夫妻宫判断

情况 婚姻状况
夫妻宫临用神 夫妻关系好，逢生大吉
夫妻宫临用神受制 婚姻不顺，两次受制二婚
夫妻宫临忌神 婚不顺，逢生二婚
夫妻宫临忌神逢制 吉，月制早婚，时制晚婚

3. 财星判断（男命）

情况 婚姻状况
财星为用神临坐下 大吉之婚，逢生妻贵而富
用财财弱而受制 婚姻有波折，妻无助
喜财见财 妻能干
忌财无财 妻得助
喜财无财 看宫位决定

4. 官星判断（女命）

情况 婚姻状况
官星为用神临坐下 大吉之婚，逢生夫贵而富
用官官弱而受制 婚姻不顺，夫无能
喜官见官 夫能干
忌官无官 夫得助
喜官无官 看宫位决定

5. 情义判断

情况 说明
男财星、女官星为用近日主 有情，夫妻关系好
男财星、女官星为用不靠近日主 无情，关系不好

6. 食伤与婚姻

情况 结果
食伤旺而透出为忌 二婚
食伤旺而透出为用 晚婚

7. 配偶长相

情况 结果
男女星宫为用 配偶长相漂亮
男女星宫为忌 配偶长相平常

四、实例分析

案例一：从强为官

分析：

  • 从强午未化火
  • 从强忌官无官

结论：为官。

案例二：发财百万

分析：

  • 身弱用印比
  • 财不制印

结论：身旺地发财（96-98年发财百万）。

案例三：印旺用财

分析：

  • 印旺用财
  • 财透藏全

结论：富。

案例四：女强人

分析：

  • 身弱用印
  • 局中见财，财不制印
  • 组合有情
  • 伤官旺得印制

结论：伤官佩印格，94-98一路发财。

案例五：贫命

分析：

  • 身旺财弱
  • 财无生

结论：贫命。

案例六：晚婚

分析：

  • 八字从弱
  • 食伤为用神
  • 旺而透

结论：虽为用亦主晚婚。

案例七：早婚

分析：

  • 身弱用印
  • 夫妻宫临忌神
  • 月制吉

结论：早婚。

案例八：二婚

分析：

  • 从弱
  • 财泄己土发挥坏作用
  • 夫妻宫未土用神制用神亥水

结论：婚不顺，二婚。

五、正偏财运用总结

财富判断要点

要点 内容
用财见财逢生 大富
忌财无财 富
忌财见财无制 贫
财发挥好作用 富

婚姻判断要点

要点 内容
星宫同参 结合财星和夫妻宫
用神临宫 婚姻好
忌神临宫 婚姻不顺
食伤旺透 晚婚或二婚

总结

正偏财类象的核心

正偏财类象的核心：财星代表财富和婚姻（男命），通过财星的喜忌和作用关系可以判断贫富状况、婚姻吉凶。

学习要点

  1. 理解正偏财的基本含义
  2. 掌握财富判断的方法
  3. 掌握婚姻判断的方法
  4. 理解星宫同参的重要性
  5. 多看案例，积累经验
