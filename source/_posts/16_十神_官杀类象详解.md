## 二、工作类象

### 1. 工作判断原则

通过官星的作用判断工作情况。

### 2. 工作判断细则

| 情况 | 工作状况 |
|------|----------|
| 用官见官 | 有工作，逢生为官 |
| 忌官无官 | 为官信息 |
| 忌官见官官弱逢制 | 为官，二次逢制大官 |
| 喜官无官 | 没有工作或工作不好，重者官灾、官非 |
| 忌官见官，官旺或逢生 | 官非、灾，牢狱 |

### 3. 工作判断表

| 组合 | 结果 |
|------|------|
| 用官+见官+逢生 | 为官 |
| 忌官+无官 | 为官信息 |
| 忌官+见官+弱+受制 | 为官 |
| 喜官+无官 | 无工作或官灾 |
| 忌官+见官+旺 | 官非、牢狱 |

### 4. 作用关系判断

| 作用关系 | 结果 |
|----------|------|
| 官发挥好作用 | 工作好为官 |
| 官发挥坏作用 | 工作不好无官，甚者为灾、牢狱 |

## 三、女命婚姻类象

### 1. 婚姻判断原则

女命婚姻看官星，但要结合宫位分析。

> ⚠️ 单看官星并不全面。官星发挥好作用，可为官，工作好，但不一定婚姻就好。

### 2. 星宫同参

看六亲的信息，宫位的作用要大于六亲的作用，最完善的看法应该是"**星宫同参**"。

### 3. 婚姻判断细则

| 情况 | 婚姻状况 |
|------|----------|
| 官为忌神，局中见官 | 婚姻不顺 |
| 官为忌神，夫妻宫临忌神受生 | 二婚信息 |
| 官为忌神，夫妻宫临用神逢制 | 二婚信息 |
| 食伤旺而透干为喜用 | 晚婚 |
| 食伤旺而透干为忌 | 婚姻不顺 |

### 4. 无官看食伤

| 情况 | 结果 |
|------|------|
| 无官看食伤 | 食伤旺而透干为喜用，晚婚 |
| 食伤为忌 | 婚姻不顺 |

## 四、实例分析

### 案例一：从弱见官贵

> 📝 **八字**：乾造：甲申 壬申 己丑 丙寅

**分析**：
- 己土月令无根印
- 坐下丑土两次被制从弱
- 从格喜官制
- 局中见官贵
- 局中有伤官不制官

**结论**：组合有情，为官。

### 案例二：从强忌官

> 📝 **八字**：乾造：丁丑 乙巳 己卯 庚午

**分析**：
- 八字印比旺
- 坐下官星两次被泄
- 从强忌官
- 官来生印应吉

**结论**：贵命。

### 案例三：官印相生

**分析**：

**结论**：官印相生贵。

### 案例四：从强无官

**分析**：
- 从强忌官

**结论**：大官。

### 案例五：身弱用印

### 案例七：官灾例
