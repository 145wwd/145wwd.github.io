## 一、体用的基本关系

### 1. 阴阳对应

体与用的关系是阴向阳、阳求阴的对应关系。

| 体 | 用 |
|----|----|
| 阴 | 阳 |
| 被动 | 主动 |
| 承载 | 发用 |
| 静态 | 动态 |

> 💡 就像谈对象的第一条件是不能把性别弄错，只有找对体用配对关系，才能正确推断它们之间的相互作用问题。

### 2. 为什么要找体用？

| 原因 | 说明 |
|------|------|
| 配对关系 | 明确谁与谁发生作用 |
| 作用方向 | 确定作用的方向性 |
| 结果分析 | 分析配对后的结果 |

---

## 二、碗喻详解

### 1. 碗的体用关系

**以碗为例**：

| 层面 | 内容 |
|------|------|
| 体 | 碗本身（属阴） |
| 用 | 碗的用途（属阳） |

**碗的用**：
- 装水 → 水碗
- 装汤 → 汤碗
- 装粥 → 粥碗

### 2. 体用关系的特点

**当碗没装东西时**：
- 没发生相应的体用关系
- 碗的价值没有体现

**当碗装了东西时**：
- 体与用融为一体相互作用
- 体现事物的意义与价值

> 💡 只有当"体与用"融为一体相互作用时，才能体现事物的意义与价值。

### 3. 一体多用

| 现象 | 说明 |
|------|------|
| 同一个碗 | 可以装水、装汤、装粥 |
| 因用得名 | 水碗、汤碗、粥碗 |
| 一体多用 | 一个体可以应多个用 |

**结论**：属阴的碗会因为用的不同而得名不同，这个名是因它的用而命名的，这也是体用关系中，**一体可以应多用**的原理。

---

## 三、阴阳三原则

### 1. 什么是阴阳三原则？

碗肯定不能，也装不下与它不配对的东西，如装不了汽车飞机。

**原因**：
- 汽车或飞机与碗不在同一范畴里
- 不符合阴阳的三原则

### 2. 阴阳三原则的内容

| 原则 | 内容 |
|------|------|
| 同范畴 | 体用要在同一范畴 |
| 相配对 | 体用要能够配对 |
| 可作用 | 体用要能够相互作用 |

## 五、体用关系的哲学意义

### 1. 现实生活的体用

**同样一件事，大家一起观察，但得出的体会与感受却不同的**：

| 层面 | 内容 |
|------|------|
| 事 | 体 |
| 人 | 用 |
| 价值观不同 | 体会感受不同 |

**原理**：以事为体，而以人为用。同样的事，对应到每个人身上，因为价值观不同所以体会感受不同，故才有不一样的反映。

### 2. 体用关系是阴阳哲学的认识论

体用关系同时也是**阴阳哲学的认识论**之一：

| 哲学层面 | 体用关系 |
|----------|----------|
| 本体论 | 体是本质，用是表现 |
| 认识论 | 通过用来认识体 |
| 方法论 | 分析体用来断事 |
