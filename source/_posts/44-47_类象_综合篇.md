## 一、类象的重要性

### 1. 理论与实践的联系

类物取象更多的是为了应用实践服务的，因此使它具有**理论与实践相互联系的特殊作用**。

### 2. 纵向与横向结构

| 结构 | 特点 | 比喻 |
|------|------|------|
| 基础理论（纵向） | 由点到线 | 丝线 |
| 类物取象（横向） | 点对点 | 珍珠 |

### 3. 生命之链

只有当通过作用关系把这些珍珠串连起来，才能结出**一条完整的生命之链**。

### 4. 天干与地支

| 类型 | 代表 |
|------|------|
| 天干 | 多为外表 |
| 地支 | 多为内脏、体内 |

> 💡 有时通用，例如，卯是乙的本气，甲是寅的透显。

## 二、五行类象

### 1. 五行木

#### 基本属性

| 属性 | 内容 |
|------|------|
| 方位 | 东方 |
| 季节 | 春天（寅卯辰月） |
| 人体 | 肝胆、四肢、毛发、神经系统等 |
| 性质 | 曲直向上，有条达、恻隐、善良、仁慈、倔强之性 |
| 五味 | 主酸 |
| 五常 | 主仁 |
| 五色 | 绿色 |

#### 木的特征

- 于季节为春，具有生发、生长的特征
- 木主仁，于人代表仁义，长寿
- 木过旺为性格倔强

#### 木与身体

| 部位 | 说明 |
|------|------|
| 肝胆 | 八字木旺多，肝胆功能亢盛 |
| 毛发 | 毛发粗浓 |
| 神经系统 | 在天为风，在地主木 |

#### 木的疾病

当命局、大运、流年，木过旺火不及，或以木为忌，会有**肝胆之疾病**。

| 属性 | 内容 |
|------|------|
| 方位 | 南方 |
| 季节 | 夏天（巳午未月） |
| 人体 | 心、眼目、心脏心血系统等 |
| 性质 | 火热、向上 |
| 五味 | 主苦 |
| 五常 | 主礼 |
| 五色 | 红色 |

#### 火的特征

- 于季节为夏天，具有火热、向上的特征
- 火主礼，与人代表文明，好礼，文采好
- 性格还代表热情、奔放
- 八字火旺的人，多热情
- 火的伤官有感染力

### 3. 五行土

| 性质 | 养育、包容、慈爱 |
| 五味 | 主甜 |
| 五常 | 主信 |

#### 土的特征

- 于季节是四季之末，具有养育、包容、慈爱的特征
- 土主诚信，于人代表诚实、信用、忠厚、朴实
- 八字土过旺则实在、老实、固执、笨拙，不灵活，甚至愚钝

| 部位 | 说明 |
|------|------|

- 土过虚则薄情寡义，脾胃不好
- 八字湿土多，则容易得皮肤病、湿疹等
- 木克土则皮肤容易留下伤疤，胃肠不好

## 三、干支类象

### 1. 天干类象

#### 甲木

《滴天髓》：**甲木参天，脱胎要火。春不容金，秋不容土。火炽乘龙，水宕骑虎。地润天和，植立千古。**

| 属性 | 内容 |
|------|------|
| 阴阳 | 阳木 |
| 特点 | 刚硬，大树之木 |
| 代表 | 首领（十天干第一），班长、头目，高直 |
| 身体 | 头 |
| 颜色 | 绿 |

**甲木在年干或时干**：头的信息更加明显（日支、月支）。

**八字中木旺**：主个子高。

**甲木火受泄太过**：多患神经衰弱，容易得精神病。

**生于寅卯月**：有水生助可成参天之木，身旺用火泄为宜。

**初春木嫩（寅月）**：一般喜辰土培根，最畏金来克害；若春木无水则枯，反不宜见火（特别是卯月）。

**夏天**：有水要用水。

**夏天、秋天**：无水为干木，干木不怕火。

**冬天**：忌水太旺，八字多差，木浸在水里根容易腐烂。

**物象**：大树、大路、桥梁、栋梁、高楼、大厦、闹市、楼梯、电线杆、粗壮直立之物等等。

**身体部位**：头、胆、毛发、指甲、上肢、神经系统等。

#### 乙木

《滴天髓》：**乙木虽柔，割羊解牛。怀丁抱丙，跨凤乘猴。虚湿之地，骑马亦忧。藤萝系甲，可春可秋。**

| 属性 | 内容 |
|------|------|
| 阴阳 | 阴木 |
| 特点 | 阴柔，缠绕，如藤萝，花草，小树 |
| 颜色 | 青绿 |

**生于寅卯月**：宜水不宜火，宜结党会局，类序成势，有水见火也吉。

**夏天**：如禾苗喜水。

**秋天**：如庄稼喜火。

**乙木弱，见甲木**：藤萝系甲，为依靠。

**水旺**：则木漂，多流动，不稳。

**在天为风，在地上为树木**，因此称它为阴木。

**物象**：藤萝之木、为花、草，多而旺或见亥、未则为山林、果园、公园、绿地、草坪、花店、出入口、栏杆、围栏、篱笆等，见卯为门窗。

**身体部位**：肝、项、四肢、毛发、指甲、神经系统等。

#### 丙火

《滴天髓》：**丙火猛烈，欺霜侮雪。能煅庚金，逢辛反怯。土众成慈，水猖显节。虎马犬乡，甲木若来，必当焚灭。**

| 属性 | 内容 |
|------|------|
| 阴阳 | 阳火 |
| 特点 | 太阳火，正直、显赫，文明、威严 |
| 颜色 | 红 |

**春天**：温暖。

**夏天**：炎热，急躁，喜见水，见辰土。

**秋天、冬天**：照耀万物。

**特点**：有博爱之心，丙火旺的人，一心想做大事业，有出人头地、做领导的欲望多强烈。

**形如太阳**：怕过旺木衰水则焚烬而亡。故丙火要抑制其性，不可以助其威。

**在天上闪烁，普照大地**：在天上是太阳是闪电，在地上是炉火是冶炼之火，因此称之为阳火。

**属于太阳火，有文明之象**：大凡六丙日的人，出生在冬夏不如春秋好，这是因为春阳有暖万物之功，秋阳有烘燥万物的作用，冬阳阴晦，夏阳炎热之故。

**物象**：太阳、大火、冶炼场、剧场等等。

**身体部位**：小肠、肩、眼目、心思系统等。

#### 丁火

《滴天髓》：**丁火柔中，内性昭融。抱乙而孝，合壬而忠。旺而不烈，衰而不穷，如有嫡母，可秋可冬。**

| 属性 | 内容 |
|------|------|
| 阴阳 | 阴火 |
| 特点 | 形如烛灯，有旺而不烈衰而不竭之性 |
| 颜色 | 红 |

**衰时最宜甲乙生助**。

**灯头火，神秘，闪耀**：力量不如丙火，见丙火则光辉被夺。

**是神秘之星**：为用时，职业大都与宗教、术数有关。大六壬中有丁马，即是丁有动的意思、如灯头之火，在现在也为网络、电子。

**是万物精气，呈文明的气象**：在天上是闪烁的星星，在地上是忽明忽暗的灯火。当太阳丙火落下时，于是星光就在天空显现出来，灯火也会明亮起来。

**属于阴火，只有到了晚上才会辉煌灿烂**：凡是日干为丁火的，喜欢生在秋天和夜里，这是因为秋天和夜里正是星光当令的时候。

**物象**：星火、星光、灯火、蜡烛、火把、火炬、网络；引申移动、香火等等。

**身体部位**：心、眼目、心脏心血系统等。

#### 戊土

《滴天髓》：**戊土固重，既中且正。静翕动辟，万物司命。水润物生，火燥物病。若在艮坤，怕冲宜静。**

| 属性 | 内容 |
|------|------|
| 阴阳 | 阳土 |
| 特点 | 如大地，旷野 |
| 颜色 | 黄 |

**在春、秋比较好（不包含戌月）**：为薄土，春薄土生机勃勃，喜丙丁火，其弱时，最爱丙火太阳普照。

**同丙火有相似之处，怕旺**：旺了没什么贵气，生于夏天的人很少能有贵格，但有所去的忌神就好，如水为忌神去忌神，见水才能富，见戌土通根则顽固。生于冬天更喜见丙火。

**怕火生过旺而无水，这样凶**。

**在中间，但也分散到东西南北四角**：在天上是雾，在地上是山峦，因此，称之为阳土。

**日干为戊土的人，最好四柱带水**：这样就是霞水相映，如年干月干逢见癸水，那就更好。

**物象**：山、陆地、引申为山岭、堤坝、墙堰、寺观、沙石场、百货部、仓库、停车场等等。

**身体部位**：皮肤、肌肉、鼻面、胃、肋、消化系统等。

#### 己土

《滴天髓》：**己土卑湿，中正蓄藏。不愁木盛，不畏水狂。火少火晦，金多金光。若要物旺，宜助宜帮。**

| 属性 | 内容 |
|------|------|
| 阴阳 | 阴土 |
| 特点 | 田园之土 |

**对四季并不敏感，一般衰则喜禄，旺则喜金水**。

**丁为己土之禄**：多数己土多喜丁火或午禄。

**生在春天喜丙火照耀，喜丁点水，喜午禄**。

**己土生夏天喜壬甲齐透，透甲见壬，财官为用**。

**无丙无辛兼无甲，一盘皆空**。

**生秋天一般都喜火**。

**生冬天，冬月天寒地冻，先取丙火，次用甲木，丙甲齐透乃杀印格**。

**在中央，是天的元气，地上的真土**：清气上升，冲合天地之气，浊气下降，生育万物，因此称之为阴土。

**性质类属于天上的云**：己土位在酉，在兑方，兑的意象是泽，天上适时降雨，山泽蒸气为云，所以己土为云。

**物象**：田地、田园、土薄之地、云、平坡、平原等等。

**历史典故**：孔子"问礼于老子"，老子对孔子说，不要一天到晚总是充满斗志想要成就功业，计划着将来如何如何，这样其实无益于自身，并且也恐怕活的不长久。孔子听了他的话，说"吾今日见老子，其犹龙邪。"龙可以"乘风云而上天"，年轻的孔子，认为老子深不可测，高不可攀。

**故宫角楼**：并不是在角落地方就叫做角楼，因为古代建筑都是用木制，为了防火，请龙王来降火，天上二十八宿中的角星对应为龙角，代表着龙，因此就请下角星下界，故名为角楼。角楼是中国的镇城之宝，角楼深刻地体现了"万物各得以生"的五行之和的目的。

#### 酉金

| 代表 | 说明 |
|------|------|
| 暗昧之神 | 化妆品、首饰、镜子、美容机构等 |

**特点**：
- 主骄傲，有条理和专横，自命不凡

**历史典故**：美国在海湾战争时，带了好多鸡到战场，这些鸡是为了防备伊拉克的毒气战。酉主暗昧，毒气是暗昧之气，鸡对毒气十分敏感；酉，在内脏代表在肺，肺开窍于鼻，故而鼻子灵。在这一点上，鸡比狗的判断力强。

## 四、类象的运用原则

### 1. 共象原则

八字中的某一字既有干支之象，又有宫位之象，还有十神之象，但它们如出现基本相同或相近的意思，那就可以确定该象了。

**例一**：乙木日主，时上见丁火
- 以丁干之象有眼睛的意思
- 以时宫位则也有眼睛之意
- 以食神星则有窍的意思
- 眼睛也是窍

**结论**：此丁火可以当眼睛看，即诸象有其共性，为共象原则。

**例二**：辛日主见癸水
- 癸有智慧的意思
- 食神有思想的意思
- 两者意思接近

**结论**：可以将此癸当思想、头脑看。
