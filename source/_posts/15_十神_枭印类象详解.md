## 一、枭印的基本含义

### 1. 正印与偏印

| 类型 | 含义 |
|------|------|
| 正印 | 文凭、母亲、身体、慈善 |
| 偏印（枭神） | 艺术、宗教、玄学、孤独 |

### 2. 枭印的共同特点

| 特点 | 说明 |
|------|------|
| 生我之神 | 对日主有保护作用 |
| 代表文书 | 文凭、合同、证件 |
| 代表母亲 | 及母亲同辈中人 |
| 代表身体 | 健康状况 |

## 二、文凭类象

### 1. 文凭判断原则

**印在命局中发挥好的作用有文凭。**

### 2. 文凭判断细则

| 情况 | 文凭高低 |
|------|----------|
| 用印见印，印旺 | 文凭高（逢生不受制更高） |
| 用印局中无印 | 看食伤（两者为忌无文凭） |
| 忌印局中无印 | 有文凭 |
| 忌印局中见印旺 | 无文凭（受制二次有文凭） |
| 用印见印，印一次受制 | 再考得文凭 |
| 用印见印，印两次受制 | 无文凭 |
| 用印印星遥隔而不透 | 低文凭 |
| 天干用印而无根，又受制 | 无文凭 |

### 3. 文凭判断表

| 组合 | 结果 |
|------|------|
| 用印+印旺+逢生 | 高文凭 |
| 用印+印旺+受制 | 中等文凭 |
| 用印+无印 | 看食伤 |
| 忌印+无印 | 有文凭 |
| 忌印+印旺+无制 | 无文凭 |
| 忌印+印旺+二次受制 | 有文凭 |

## 五、实例分析

### 案例一：有文凭

> 📝 **八字**：乾造：庚戌 己卯 丙戌 庚寅

**分析**：
- 丙火生卯月
- 时支临印
- 干无引发身弱用印
- 用印局中见印

**结论**：有文凭（大专毕业）。

### 案例二：本科生

> 📝 **八字**：乾造：甲寅 丙子 己丑 辛未

**分析**：
- 己生子月失令
- 得丑未根居日时弱
- 身弱用印
- 得丙火生

**结论**：有文凭（本科生）。

### 案例三：博士生

> 📝 **八字**：坤造：癸丑 癸亥 庚午 丙戌

**分析**：
- 身弱用印
- 有印星相生

**结论**：有文凭（博士生）。

### 案例四：从弱有文凭

> 📝 **八字**：乾造：辛亥 己亥 丁未 辛亥

**分析**：
- 从弱忌印
- 局中无印

**结论**：有文凭。

### 案例五：无文凭

> 📝 **八字**：坤造：己未 丁丑 甲午 丁卯

**分析**：
- 甲生丑月失令
- 得时支卯根又被午火泄
- 从弱无印吉
- 伤官被制凶

**结论**：无文凭，且不聪明（卯木生午火凶的组合）。

### 案例六：身旺忌印

> 📝 **八字**：乾造：癸巳 辛酉 癸亥 甲寅

**分析**：
- 身旺忌印
- 见印

**结论**：无文凭。
