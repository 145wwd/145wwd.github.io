## 一、偏财应用

### 1. 偏财在四柱

| 位置 | 应用 |
|------|------|
| 年柱偏财 | 易发他乡，但不利父 |
| 月柱偏财 | 多同正财 |
| 日坐偏财 | 爱偏房，不爱正室 |
| 时干偏财支为比劫 | 破财伤妻，先富后贫 |
| 时为偏财 | 子女慷慨，善理财当家 |

### 2. 偏财特性

| 特性 | 说明 |
|------|------|
| 他乡成家 | 他乡成家立业而富 |
| 身弱偏财多 | 易被父亲拖累 |
| 偏财多 | 情人多，好赌好色，为情耗财 |
| 不利子息 | 旺财不旺人，财多耗身，难有儿子有女儿 |
| 投机取巧 | 好投机取巧钻营，虚荣心强，贪图享受 |
| 透干 | 父掌家权 |
| 远方经营 | 利远方经营，从商致富，易金屋藏娇 |

### 3. 偏财组合

| 组合 | 结果 |
|------|------|
| 身旺财旺官旺，行官运 | 发财 |
| 身旺财旺官旺，行财运 | 升官 |
| 偏财透干 | 怕比劫破财，克父克妻 |
| 年上偏财 | 必发外帮、外省、外乡 |
| 偏财怕刑冲比劫 | 身旺逢财运大发 |
| 偏财透干 | 为人大方，仗义轻财，好酒色，不爱正妻爱妾 |
| 偏财坐长生 | 父亲长寿，财旺又逢长生，父为大富之人 |
| 偏财坐沐浴 | 因色耗财，父或妻风流 |
| 偏财会建禄 | 父或妻妾发达 |
| 偏财坐七杀 | 父漂泊异乡，为色破财 |
| 偏财坐比劫 | 有感情风波但不离婚，父或妻妾多病 |
| 偏财坐印 | 易克母 |
| 偏财坐官 | 名利双收，得青年之助，子孙发财 |
| 偏财坐羊刃 | 父和妻妾必病，易有血光之灾 |
| 日旺财弱 | 喜食伤泄身生财 |
| 偏财与将星同柱 | 掌财政大权 |

## 二、正官应用

### 1. 正官特性

| 特性 | 说明 |
|------|------|
| 一位清粹或为用神 | 光明正大，品行端正，为人公道，严肃有威 |
| 官多为虚名 | 不利公职，女命官多易克夫 |
| 有官又见伤 | 破格，凡事不满，事业不顺，宜单干不宜公职 |
| 官星逢伤 | 易有官司牢狱之灾 |
| 官杀无制 | 易犯牢狱官司，好酒好色，固执，不贫则夭 |

### 2. 正官组合

| 组合 | 结果 |
|------|------|
| 四天干自坐长生 | 定居相位 |
| 劫财逢官星 | 主执兵权，助圣明，国家得安宁 |
| 一官清贵 | 四海扬名 |
| 三杀遇三奇（财官印） | 得地，为阵上将军 |
| 年柱纳音五行生在旺地，日或时干克年干 | 官贵之命 |
| 官多身弱 | 怀才不遇 |
| 戊己土多又在旺地 | 主儿孙为农业或军需之官 |
| 年柱纳音克日柱纳音 | 主有兵权 |
| 日柱或时柱纳音克年柱纳音 | 叛逆，主犯法有凶灾 |
| 女命官星被他干合去 | 易有婚变 |
| 正官坐长生、沐浴、冠带、临官或帝旺 | 官位当高，宜任公职 |
| 官星坐衰、病、死、墓或绝地 | 不宜出任公职，防官灾，不利子女 |
| 官星坐财 | 得妻或女人相助而升官 |
| 官坐印地 | 学业佳，有名声，当文官 |
| 官坐比劫 | 克兄弟 |
| 正官坐伤官 | 不利公职，防仇人相害 |

### 3. 正官在四柱

| 位置 | 应用 |
|------|------|
| 年上官星又为用神 | 得父母之恩，父母有贵或名气，学业佳，生于富贵之家，继承和掌管家业（长子） |
| 年上正官将星 | 为人正直，父母权力显赫 |
| 月上正官 | 兄弟荣耀，为正气官星，正直负责 |
| 日上官星 | 得妻之助，女命得贵夫，遇事应变能力强，指挥能力强 |
| 时柱官星 | 子女聪明而厚道，逢死绝者不吉 |

### 4. 官运判断

| 情况 | 结果 |
|------|------|
| 流年大运逢羊刃、劫杀、将星等，组合成官局 | 官大或升官 |
| 四柱中有伤官七杀，大运流年是羊刃将星 | 升官，官职高 |
| 食神带刃坐官星 | 职高一品 |

## 三、偏官（七杀）应用

### 1. 七杀特性

| 特性 | 说明 |
|------|------|
| 职业 | 军警武职、公检法移民局、黑社会 |
| 得食神制 | 为贵，有实权又显威，计谋多，有智有谋 |
| 无制 | 为贱、为小人 |
| 受制太过 | 身旺杀弱又逢制，反为卑鄙，无恶不作 |
| 行事 | 果断豪放直爽，但易叛逆，偏激霸道 |
| 多逢 | 少乐多忧，易犯牢狱之灾 |

### 2. 七杀组合

| 组合 | 结果 |
|------|------|
| 日弱杀旺怕逢财 | 凶 |
| 日旺杀弱喜见财 | 吉 |
| 日弱又官杀混杂 | 危机四伏，灾难多，一事无成，易流入邪恶小人之流 |
| 日主旺，杀印同透 | 名播四海 |
| 日主旺，财杀同透 | 主武职 |
| 官杀多而混杂 | 胆怯无能而易冲动 |
| 七杀空亡无救 | 易失权位不宜公职 |
| 七杀伤官同透 | 易叛逆，及流于黑社会凶手 |
| 日旺，杀刃同柱 | 权威显赫，掌生杀之权 |
| 七杀坐墓或流年大运逢墓地 | 易犯官灾牢狱之苦，而且无后嗣 |
| 七杀坐印 | 文武双全，有权有威，功名显赫，事业发达 |
| 身旺甲木为七杀遇死伤 | 有头无脑，死脑筋不开窍神经病 |

### 3. 七杀在四柱

| 位置 | 应用 |
|------|------|
| 年上七杀 | 非长子，为忌神生贫贱之家，有奴欺主之事，年杀月伤兄弟难圆 |
| 月上七杀 | 好抛头露面，爱出风头 |
| 日支七杀 | 配偶性烈刚强粗鲁，如杀弱则配偶多无能，或身体不佳 |
| 时柱七杀 | 子女多不贤孝，若有制可生贵子 |

### 4. 七杀注意事项

| 情况 | 结果 |
|------|------|
| 七杀在日支，又干上透出无制，若日主弱 | 其祸不可言 |
| 身弱官杀旺之人，应去印比之方工作 | 否则易有官灾、病灾和事业不顺 |
| 身弱七杀坐财 | 因财或女人妻子致灾 |

## 五、偏印应用

| 特性 | 说明 |
|------|------|
| 无食神 | 为偏印，有食神为枭神 |

| 偏印者 | 精明强干，固执孤独，主观刻薄善变，多学少成 |
| 身旺再逢 | 大事干不了小事不愿干，不自量力，虚伪，耍小聪明 |
| 为喜用神 | 多才多艺 |

### 2. 偏印组合

| 组合 | 结果 |
|------|------|
| 身弱最宜偏印生身 | 变弱为强；但忌偏财克，食神耗 |

| 正偏印同透 | 除正业外，还喜偏业 |
| 身旺有偏印，四柱中财官旺同透 | 富贵 |
| 四柱中干支皆为偏印 | 离祖谋生，易失权贵，易招盗贼，且身体不好 |
| 偏印多 | 不是义母多，就是继母多 |
| 四柱中有偏印 | 不自由，受他人制约 |
| 四柱中有偏印，见官杀混杂 | 为多成多败之命 |
| 偏印喜伤官配 | 正印喜食神配 |
| 枭神主病伤灾之事 | 偏印易伤头、手指、脚 |
| 四柱中有偏印为内夺 | 大运流年见枭为外夺，灾自外来 |

### 3. 偏印在四柱

| 位置 | 应用 |

| 日支偏印 | 男不得良妻，女不得良夫，逢空亡母缘差，女命易产 |

| 四柱中有偏印 | 最怕枭食夹日，不犯牢狱之灾则痛苦难言 |

| 身弱财多 | 喜四柱有比劫帮身，羊刃帮身，印生身，禄来护身 |

| 身弱财多 | 喜在比劫、印绶的方位居住工作 |
| 身弱财多 | 应从事有关助身的行业或生意 |
| 身弱财多 | 忌四柱中有官星，怕走财运官运 |
| 身旺财少 | 怕见比劫，克父克妻；又怕见羊刃、印、禄 |
| 身旺财少 | 忌流年大运走羊刃禄印地 |
| 身旺财少 | 喜去有财的方位工作发财 |
| 身旺无财 | 除以上的方法外，尚应找命中为财星的男女为妻为夫为友 |

| 四柱中正财偏财多而混杂 | 愚笨 |

| 正财坐建禄 | 大贵大富之人 |

**十神应用的核心**：通过十神在四柱中的位置、组合、旺衰来判断吉凶祸福。不同的十神代表不同的人、事、物，通过分析十神的作用关系可以推断命运。

1. 熟记各十神的基本含义
2. 掌握十神在四柱的不同应用
3. 理解十神的组合关系
4. 多看案例，积累经验
